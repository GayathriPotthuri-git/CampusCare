# 🏛️ Campus Care - Issue Reporting System

A modern, real-time campus issue reporting system that automatically routes complaints to the appropriate authorities with detailed profiles and instant notifications.

## ✨ Features
- 🎨 Modern, animated landing page
- 📝 Smart complaint submission with auto-routing
- 👥 Detailed authority profiles with photos and contact info
- 📊 Real-time dashboard with live statistics
- 🔔 Instant notification system
- ✅ Status tracking (pending/resolved)
- 📱 Fully responsive design

## 🚀 Quick Start

1. Install dependencies:
```
npm install
```

2. Start the server:
```
npm start
```

3. Open your browser and visit:
- **Landing page**: http://localhost:3000/landing.html (START HERE!)
- Report issue: http://localhost:3000
- View authorities: http://localhost:3000/authorities.html
- Dashboard: http://localhost:3000/dashboard.html

## 📄 Pages Overview

1. **Landing Page** (`landing.html`) - Beautiful hero section with animated floating cards
   - Click on Maintenance, Electrical, or IT Support cards to see dedicated pages
   
2. **Category Pages** (`category.html`) - Dedicated pages for each service type
   - Shows common issues handled
   - Department head information
   - Recent reports in that category
   
3. **Authorities Page** (`authorities.html`) - Meet all department heads
   - Photos, contact info, experience, availability
   
4. **Report Issue** (`index.html`) - Submit complaints
   - Auto-routes to correct department
   
5. **Dashboard** (`dashboard.html`) - View all complaints
   - Live statistics
   - Mark as resolved

## Categories

- Electrical issues
- Water supply
- Network/IT issues
- General maintenance
- Other
